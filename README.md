# InsightHR: Performance Predictor 🚀

**An AI-Powered Decision Support System for Modern HR Teams.**

InsightHR is an advanced analytics dashboard designed to predict employee performance using machine learning classification. Built with a focus on interpretability and actionable insights, the platform allows HR professionals to simulate model behavior, visualize complex talent data, and receive AI-driven managerial recommendations.

![InsightHR Preview](https://picsum.photos/seed/insight-hr/1200/600)

## 🌟 Key Features

### 1. Interactive Algorithm Simulator
*   **Dynamic Feature Weighting**: Adjust the relative importance of tenure, review scores, training hours, and project completion in real-time.
*   **Live Recalculation Engine**: See how your "talent hypothesis" affects accuracy, precision, and recall metrics across the entire workforce instantly.
*   **Non-Linear Interaction Simulation**: The model accounts for sophisticated employee behaviors, such as the synergy between high engagement and intensive training.

### 2. Deep-Dive Talent Analytics
*   **Skill Fingerprinting**: Visualize individual employee potential across five dimensions (Technical, Communication, Leadership, Reliability, Innovation) using interactive Radar Charts.
*   **Workforce Distributions**: Scannable bar and pie charts showing departmental participation and performance splits.
*   **Risk & Confidence Mapping**: Every prediction is accompanied by a confidence score and a derived "Plateau Risk" assessment.

### 3. Gemini AI Integration
*   **Managerial Insights**: High-level, context-aware recommendations generated by Google's Gemini AI.
*   **Actionable Advice**: Tailored suggestions on whether an employee needs specific training, is ready for a promotion, or requires re-engagement strategies.

### 4. Advanced "Mission Control" UI
*   **Professional Dashboard Layout**: A sleek, information-dense interface built for decision-makers.
*   **Real-time Model Evaluation**: Detailed Model Metrics view including a Confusion Matrix to audit AI reliability.
*   **Fluid UX**: Smooth tab transitions and reactive components powered by Framer Motion.

## 🛠️ Tech Stack

*   **Frontend**: React 19, TypeScript
*   **Styling**: Tailwind CSS 4.0
*   **Visualizations**: Recharts (Radar, Bar, Pie charts)
*   **AI Engine**: Google Gemini API (`@google/genai`)
*   **Animations**: Motion (formerly Framer Motion)
*   **Utilities**: Lucide React, clsx, tailwind-merge
*   **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   An API Key for **Google Gemini AI** (Get one at [Google AI Studio](https://aistudio.google.com/))

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/insight-hr-predictor.git
    cd insight-hr-predictor
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up environment variables**
    Create a `.env` file in the root directory:
    ```env
    GEMINI_API_KEY="your_api_key_here"
    ```

4.  **Run the application**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📈 Machine Learning Logic

The application uses a simulated **Random Forest / Logistic Regression** hybrid architecture. Unlike traditional static dashboards, InsightHR calculates:
*   **Precision**: Ensuring we don't falsely identify high performance.
*   **Recall**: Crucial for detecting employees who need intervention before they disengage.
*   **F1 Score**: A balanced metric for overall data-driven decision quality.
